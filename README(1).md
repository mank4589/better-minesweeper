
# MINESWEEPER

Welcome to my implementation of the classic Minesweeper game. This project is built on python and uses tkinter library for its gui.


## Features

- Highscore leaderboard
- Fullscreen mode
- Timer
- Flag suspected cells



## How To Play

1. Enter your name.
2. Select Difficulty: Choose your preferred difficulty level.
3. Revealing Cells: Left-click on a cell to reveal it. If it's a mine, the game is over.
4. Flagging Cells: Right-click on a cell to flag it as a suspected mine.
5. Win Condition: Successfully reveal all cells that are not mines.
